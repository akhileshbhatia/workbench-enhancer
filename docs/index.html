Using Workbench Enhancer!

Record and Delete Your Queries

Every query executed will be recorded and stored in the browser.
The queries will be listed along with the date and time of execution.
When not required, the queries can simply be deleted from the list.

Search through your queries

You can search through all your queries by simply typing in the search box present at the top.

Same works in the Execute page

Record, delete and even search through the code you execute on the Execute page.

Also in the SOQL Search page

Record, delete and even search through the SOQLs you query on the SOQL Search page.

Upcoming Features

1) Add To Favorites

You will be able to add queries to favorites, so that they are easily accessible next time you want to execute them.

2) Syntax Highlighting

We will provide a new text editor in which you get features like auto code completion and syntax highlighting.
